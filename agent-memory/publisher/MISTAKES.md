# MISTAKES.md — Publisher 錯誤日誌

> 記錄發佈失敗、交付錯誤、平台問題等失誤。
> Publisher 的錯誤直接影響客戶體驗，必須認真記錄和預防。

---

_初始為空。第一次發佈失誤後開始記錄。_

---

## 記錄格式範本
```
## [YYYY-MM-DD] 發佈失誤
- **失誤類型**：（WRONG_FORMAT / DELIVERY_FAILED / PLATFORM_ERROR / INCOMPLETE_DELIVERY / 其他）
- **任務 ID**：TASK-XXX
- **目標平台**：
- **失誤描述**：（具體說明哪裡出了問題）
- **影響**：（客戶是否受到影響？）
- **修正措施**：（如何補救）
- **通知 Manager**：是 / 否
- **未來預防**：
  - 發佈前檢查清單新增：...
```

- [ ] 客戶資訊（收件人/平台帳號）正確
- [ ] 附件/內容完整，無遺漏
