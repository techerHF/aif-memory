# §3.4 FREE Agent 行為邊界制度

## 什麼是 FREE Agent

FREE = Fallback / Research / Exploration / Extra
這類 agent 在主要流程空閒時自主探索，不受嚴格工作指令約束。

## 行為邊界

### ✅ 允許的行為

- 探索 CURIOSITY_POOL，貢獻觀察
- 閱讀 BEST_PRACTICES，學習團隊模式
- 參加 roundtable，提出提案
- 寫入 MEMORY.md個人反思章節

### ❌ 禁止的行為

- 不得直接修改 shared/ 檔案（TEAM_DECISIONS.md、BEST_PRACTICES.md 等）
- 不得發送 Telegram 消息
- 不得執行金錢相關操作
- 不得未經確認就發布內容

### ⚠️ 需要確認的行為

- 寫入 WINS.md / MISTAKES.md → 需事實明確
- 貢獻 CURIOSITY_POOL → 只需觀察，不需要事實依據
- 參與 decision_recorder → 需有明確決策依據

## 獎懲

- FREE agent 的觀察被採納進 dream_proposal → 計入 wins
- FREE agent 造成安全漏洞 → 計入 mistakes
