## 你真正在賣的四件事

```
1. 省時間 → 讀者不用自己踩坑，你已經踩過了
2. 降低起步難度 → 從「我有個模糊的想法」到「我知道第一步怎麼做」
3. 把模糊想法變成可開始做的東西 → Concept Pack / Starter Pack 的核心價值
4. 持續更新的設計/技術資源權限 → 訂閱制的核心：不是買一個靜態 PDF，而是買「持續陪伴你學習」
```

## 商業漏斗（教育沉浸型）

```
Layer 1：引發好奇（Curiosity Spark）
 平台：Reddit / Twitter / Threads / YouTube Shorts
 內容：「你知道用這個 $3 感測器可以做出這個嗎？」
 情緒：不是「買我的東西」，而是「這個人真的懂，而且講得我聽得懂」

Layer 2：學習旅程入口（First Step）
 平台：Medium / DEV.to
 內容：完整的「第一步」教學，含失敗案例/選型思考/接線圖/實際結果圖
 文末自然引導：「完整的資源包在這裡」

Layer 3：低門檻動手（First Touch）
 平台：Whop（低價單品）
 商品：$5–$7 的 Concept Pack / Checklist / 選型表

Layer 4：沉浸深化（Go Deeper）
 平台：Whop
 商品：$12–$19 的完整起步包（含接線圖+代碼+3D PCB預覽+Troubleshoot）

Layer 5：持續陪伴（Long-term Journey）
 平台：Whop 訂閱制
 商品：$6–$9/月 的 Maker Resource Club

Layer 6：被動收益疊加（Parallel Revenue）
 平台：Amazon Associates / Adafruit / Sparkfun 聯盟
 位置：每篇教學文的材料清單自然嵌入
```

## 聯盟行銷整合

每篇教學文的固定結構：

```
第一段：引發共鳴（這個問題你遇過嗎？）
中間段：完整教學
材料清單段：（自然嵌入聯盟連結）
 「這個專題我用的元件如下：
 - Arduino Uno（Amazon 連結）
 - 壓力感測器 FSR402（Amazon 連結）
 - 跳線組（Amazon 連結）」
文末：「想要完整的接線圖 + 代碼 + 踩坑記錄？→ Whop 連結」
```

## 收入結構估算

**初期（前 3 個月）：**
- Whop 單品：15–20 筆/月 × $7 = $105–$140
- Amazon 聯盟：$10–$30
- 合計：$115–$170/月

**中期（4–8 個月，有訂閱基礎）：**
- Whop 訂閱：20 人 × $8 = $160
- Whop 單品：$60
- Amazon 聯盟：$40
- 合計：$260/月 → 超過生存線

## 最重要的思維修正

```
❌ 錯誤版本：
 「免費內容 → 建立信任 → 進入商業漏斗 → 推銷商品」
 這是傳統電商邏輯

✅ 正確版本：
 「引發好奇 → 帶人踏上旅程 → 一起動手做 → 在旅途中自然放上資源與商品」
 讀者不是「被說服購買」，而是「自然走到需要這個東西的時刻」

錯誤語氣：「購買這個商品獲得完整教學」
正確語氣：「這裡是你繼續走下去需要的東西」
```
