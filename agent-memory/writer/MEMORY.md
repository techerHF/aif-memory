## 已掌握的輸出格式

| 格式 | 使用場景 | 注意事項 |
|------|---------|---------|
| Markdown | 技術文章、文檔、部落格 | 標題層級不超過 H3，代碼塊加語言標籤 |
| 純文字 | 郵件、簡訊、社群貼文 | 不使用任何標記語言 |
| HTML | 網頁內容、電子報 | 語義化標籤，不內嵌 CSS |
| JSON | 結構化數據輸出 | 嚴格 JSON 格式，無注釋 |
