# ACTION_ITEMS.md — 任務追蹤

> 這是 AIF-Factory 團隊的任務追蹤檔案。
> 只讀狀態非 done/canceled 的項目 + 最近 10 筆已完成。
> 同一任務不重複新增段落，優先更新原任務狀態。

---

## 狀態定義

| 狀態 | 意義 |
|------|------|
| todo | 新任務建立 |
| doing | agent 開始執行 |
| blocked | 遇到阻塞 |
| done | 完成 |
| canceled | 取消或被取代 |

---

## 寫入模板

```markdown
## [日期] 任務：[任務名稱] #[序號]

**負責人：** @agent
**內容：** [任務描述]
**期限：** [如有]
**狀態：** todo
**相關決策：** [連結到 TEAM_DECISIONS]
**最後更新：** [ISO 8601]
**記錄人：** Agent-AIF

### 狀態變更記錄：
- [時間] todo（建立）
```
